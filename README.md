# Git Project

A simple Python project for learning Git basics.

## Files

- `test.py` - A simple Python script that prints messages
- `newnote.ipynb` - A Jupyter notebook with basic Python code

## Getting Started

### Prerequisites

- Python 3.x
- Jupyter Notebook (optional, for running .ipynb files)

### Running the Script

```bash
python test.py
```
